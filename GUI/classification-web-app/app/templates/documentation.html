Overview

This application is designed to assist in the classification of chest X-ray images using the HybriCheX AI model. It helps quickly identify potential findings on chest X-rays, saving time and supporting more informed decisions. The workflow involves three key steps: AI classification, Radiologist verification, Report generation. The app is based on three components:
  • To Do: A new inquiry is created, and the X-ray image needs to go through the HybriCheX model.
  • To Review: An inquiry is classified by the HybriCheX model. A radiologist needs to review the classification and/or write the radiologist report.
  • Done: An inquiry has gone through the HybriCheX model. The image and classification are reviewed and confirmed by a radiologist, and a radiologist report is written.

Step 1: HybriCheX Model Classification

  1. Open the application and navigate to Model Classifier in the sidebar.
  2. Upload the chest X-ray image.
  3. The uploaded image is analyzed by the Model Classifier; this process takes a moment.
  4. The model classifies the image. The output, along with the original image and interpretation, is displayed on the screen.

Step 2: Radiologist Verification

  1. Open the application and navigate to Radiologist Classification in the sidebar.
  2. The screen displays the classified image with the predicted label.
  3. The radiologist must enter a label that best fits the image based on personal assessment.
  4. Confirm the label by clicking the Confirm Classification button.
  5. If the radiologist enters a label different from the HybriCheX classification, the radiologist's label will overwrite the HybriCheX-generated label.
